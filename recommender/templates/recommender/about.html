About Our System

This tool is to help screen vaccinated members of the public before they attend various events in Singapore. No personal data is stored by this tool. This tool can be re-deployed to cater to various events. We use a decision tree model approach to establish the health of the participants. This model was trained with over 1 million tagged records.

The Technology

This tool is built on python and leverages on the Django framework. It incooperates sqlite to maitain and organise the data along with CSS to standardize the design of the site.
